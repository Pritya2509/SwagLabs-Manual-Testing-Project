# SwagLabs-Manual-Testing-Project

## Project Overview
This repository contains the manual testing documentation for the Swag Labs web application.  
The objective of this project is to validate the functionality of the application using manual testing techniques.

Application Tested: Swag Labs (E-commerce demo application)

## Modules Tested
- Login Page
- Products Page
- Cart Page
- Checkout Process
- Navigation Menu
- Social Media Links

## Testing Artifacts Included
The project contains the following testing documents:

1. Test Scenarios and Test Cases (TsTc)
2. Test Case Template
3. Test Data Set
4. Defect Report
5. RTM (Requirement Traceability Matrix)

## Test Case Examples
- Verify login with valid & invalid credentials
- Verify products are visible and clickable
- Verify add to cart functionality
- Verify cart page elements
- Verify checkout button functionality
- Verify social media icons functionality

- Microsoft Excel
